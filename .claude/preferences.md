# 内容偏好 — 改这个文件，不用改代码

每次 AI News 管线跑完后，Claude 会参考这个文件做精选和排序。
你任何时候说「下次多关注 X」「不要 Y」，我来更新这个文件。

## 关注方向（优先级从高到低）

1. **AI Agent / MCP / 工具链** — Agent 框架、MCP 协议、Claude Code、Cursor 等 AI 编码工具
2. **Web3 + AI 交叉** — 链上 AI、去中心化推理、加密 x AI 应用
3. **实际可用的 AI 工作流** — 自动化、效率工具、一人公司实践
4. **开源模型与推理** — 新模型发布、推理优化、部署方案
5. **AI 产品与 UX** — 好的 AI 产品设计、交互模式

## 不关注

- 纯融资新闻（除非金额特别大或涉及上面关注的公司）
- 大厂 PR 稿（没有实际产品发布的那种）
- 纯学术论文（除非有开源代码 + 实际可用）

## 信息来源权重

- AINews 邮件: 基础源，但不够及时
- Bing/HN/ArXiv: 补充深度
- GitHub Trending: 权重最高（有代码 = 有真相）
- X/Twitter: 适合捕捉新发布和争议

## 卡片风格偏好

- 语言: 大白话，像跟朋友聊天
- 深度: 不只是「发生了什么」，更要「为什么值得关注」
- 结构: 3 层 info-card（发生了什么 / 深入了解 / 为什么值得关注）+ 记住三点 + Insight Box
- 数量: 工作日 4 条 AI 新闻 + 1 条 GitHub 趋势
