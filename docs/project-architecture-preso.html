第 7 章
三条贯穿始终的
设计原则
1
生产与分发解耦
工厂产出内容资产，发布层只管平台适配。
加 YouTube Shorts？发布层加一个适配器，不动工厂代码。
2
视觉表现可插拔
颜色、字体、间距、布局模式 → 全在 brands/*.json。
品牌切换改一个 JSON 字段，设计 token 与业务逻辑彻底分离。
3
知识不依赖人，依赖系统
Skills 把经验变成可执行指令。偏好存文件，竞品自动追踪。
工具知道你要什么，不是你每次都告诉它。
